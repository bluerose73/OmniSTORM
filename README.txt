How to Use
==========

1. Double-click OmniSTORM.mlapp in the file explorer, or open it in MATLAB. No need to add any folders to PATH.
2. Click "Load Image" if the raw image is a single file; Click "Load Multi-File Image" if the image are multiple files in a folder. Supported format: TIFF and DCIMG.
3. Click "Run".
4. If the number of emitters are too many or too few, adjust the Threshold.
